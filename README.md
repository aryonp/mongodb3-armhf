# Install MongoDB 3.0.14 in armhf

As its name, automating everything that you need to do to install MongoDB 3.0.14 in armhf 

## Requirements

Tested on Ubuntu 16.04

### Installation

1. git clone https://github.com/aryonp/mongodb3-armhf.git
2. cd mongodb3-armhf/
3. sudo chmod +x *.sh
4. ./install.sh
